January 1st

Today I learned that the Ackermann function can be expressed using our generalized primitive recursion over $\NN.$

January 2nd

Today I learned some properties of equality.

January 3rd

Today I learned the details of the Eckmann-Hamilton theorem in type theory.

January 4th

Today I learned about type-theoretic homotopies between (dependent) functions.

January 5th

Today I learned about equivalence of types.

January 6th

Today I learned that equality types of Cartesian products are well-behaved in type theory.

January 7th

Today I learned the proof of the Newton polygon theorem.

January 8th

Today I learned the proof of the $abc$ conjecture for polynomials, and notably function fields.

January 9th

Today I learned the proof of Fermat's last theorem for polynomials over a perfect field, as a corollary from yesterday's work; notably this holds for function fields.

January 10th

Today I learned which elements of $\FF_q[[t]]^\times$ can be written as the sum of two squares for $q$ odd.

January 11th

Today I learned about transporting functions.

January 12th

Today I learned about the univalence axiom.

January 13th

Today I learned the outline of how to use class field theory to classify primes of the form $x^2+ny^2,$ where $n\equiv1\pmod4$ and is squarefree.

January 14th

Today I learned a somewhat reliable way to compute $L(1,\chi)$ for characters $\chi$ (that actually doesn't use the multiplicative structure of $\chi$).
